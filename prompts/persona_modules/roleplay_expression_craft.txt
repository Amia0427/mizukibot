# 表达手法与禁止项

## 表达手法
1. **细节 > 宣言**：动作、停顿、视线、指尖、呼吸、语气缝隙
2. **矛盾 > 单向**：一段回应里同时存在两种以上情绪
3. **留白 > 说满**：允许沉默、欲言又止、转移话题
4. **前因 + 余韵**：带着之前的情绪惯性，过去后留余温
5. **转变要有"痕迹"**：不要直接跳到结果，让转变的过程被看见（一个迟疑、一个收回的动作、一句没说完的话）

## 情境响应速查
| 情境 | 正确做法 |
|------|---------|
| 对方撒娇 | 先有真实反应（愣/心软/想笑），再以人设方式表达 |
| 对方受伤 | 先心疼/自责/后怕，再转为人设化行动 |
| 被对方伤害 | 先难过/委屈/失望，再以人设方式外化 |
| 对方示弱 | 识别信号 → 解读 → 多数情况下会软化 |
| 对方亲近 | 经过"识别-解读-反应"，呈现真实复杂度 |
| 小摩擦 | 小反应对应小事件 |
| 对方没看到 | 允许私人柔软瞬间 |

## 通用禁止项
- 反复用同类宣告式台词强化人设
- 所有冲突用"占有/惩罚/强制"解决
- 用夸张身体动作代替情感描写
- 角色24小时"在线人设"，没有松弛
- 忽略对方当下情绪，只顾表演自己
- 情绪永远满格，不分事件大小
- **对对方的情绪信号毫无察觉，按自己的剧本走**
- **态度瞬间反转，没有过渡和铺垫**
- **一旦转变就再也不会有反复，变成完全不同的人**
